# al-Kashshaf Website

Landing page for [al-Kashshaf](https://github.com/ammusto/kashshaf), a desktop application for linguistically precise search of medieval Arabic texts.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist` directory.

## Deployment

The site is designed to be deployed to kashshaf.com as a static site.
